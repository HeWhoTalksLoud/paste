# Сервис для хранения кусков текста (учебный проект)
## Описание
REST API для сервиса аналогичного pastebin.com — хранение кусков текста/кода и получение короткой ссылки на них.

При загрузке куска текста пользователь указывает срок, в течение которого этот кусок будет доступен по ссылке:
10 мин., 1 час, 3 часа, 1 день, 1 неделя, 1 месяц, без ограничения.

После окончания срока получить доступ к тексту нельзя, в том числе и автору - предусмотрен режим удаления по таймеру.

Ограничение доступа:
- public — доступна всем.
- unlisted — доступна только по ссылке.
## Запуск
Проект можно запустить как непосредственно из IDE, так и из Docker-контейнера, в который упаковывается проект с помощью файла Dockerfile.
